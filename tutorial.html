Instalando o e o RStudio

Para baixar o software visite o link: https://cran.r-project.org/


CRAN - Comprehensive R Archive Network


Selecione entre as opções de acordo com o seu sistema operacional (OS, na sigla em inglês).

Windows

A opção Download R for Windows abre a seguinte janela:



Os hyperlinks base ou install R for the first time redirecionam para o repositório:


R-4.0.2 for Windows

Clique em Download R 4.0.2 for Windows, ou na versão mais recente que estiver disponível.

Ao salvar e executar o arquivo o download será iniciado bastando seguir as instruções de instalação.


Por fim, é importante verificar onde os arquivos baixados estão salvos, este será o seu diretório principal!

Para os usuários de Windows, recomenda-se também a instalação do Rtools que permite a criação de pacotes. O download pode ser feito pelo link https://cran.r-project.org/bin/windows/Rtools/. Basta clicar em um dos links apresentados, de acordo com o Windows (32 ou 64-bit) instalado em seu computador.

Caso seja necessário, para descobrir se o seu sistema operacional roda em 32-bits ou 64-bits, abra o Prompt de Comando do Windows:



Copie e cole a seguinte linha de comando wmic os get osarchitecture.


Mac e Linux

Para instalar o R em computadores com o sistema operacional Mac você deverá clicar em Download R for (Mac) OS X. Na página seguinte, clique em R-4.0.2.pkg ou em outra versão mais atual.

Se você utiliza Linux o R pode ser instalado no Ubuntu diretamente pelo terminal usando o seguinte script Bash:

sudo apt-get install r-base

Download do RStudio

O RStudio é um ambiente de desenvolvimento integrado ou integrated development environment (IDE). Um IDE reúne características e ferramentas de apoio ao desenvolvimento de software com o objetivo de agilizar este processo.

Para fazer o download do RStudio, acesse: https://rstudio.com/products/rstudio/download/



Clique na opção gratuita e o site irá redirecionar para o repositório e indicar a melhor versão para o seu OS.


Funcionalidades do RStudio

Ao abrir o RStudio, inicia-se uma sessão do programa



Note que a versão e a plataforma que o programa está rodando são informadas no console:

R version 4.0.1 (2020-06-06) – “See Things Now”
Copyright (C) 2020 The R Foundation for Statistical Computing
Platform: x86_64-w64-mingw32/x64 (64-bit)

Para ampliar/reduzir os painéis, basta clicar com o cursor sobre as divisórias e arrastá-las. Também há botões de minimização e ampliação de cada painel .

Outras opções de layout dos painéis podem ser definidas em > Tools > Global Options > Panel Layout





O Panel Layout permite incluir/remover abas com demais funcionalidades, além de trocar a disposição de cada painel.

Console

A janela do Console roda os comandos e exibe seus resultados, erros e avisos.

Para limpar o console digite Ctrl+L ou clique no icone no canto direito do painel.



Source

Ao iniciar a edição de um documento, abre-se o painel Source. Este é o editor de códigos (scripts) para todos os formatos de arquivos disponíveis no R. Para iniciar um novo documento clique no botão que irá abrir o seguinte menu de opções



Dentre os tipos de arquivos disponíveis estão R Script, C++, R Markdown, Python Script, R HTML… Este tutorial está escrito em arquivos de R Markdown, com complementos de CSS e HTML, todos editados no RStudio. Para publicar o site utilizou-se o GitHub . Todos os códigos e imagens utilizados neste tutorial estão diponíveis para acesso no repositório do GitHub.

Os códigos próprios para R são criados em documentos de texto com extensão .R, os R Scripts.

Para abrir um novo R Script basta pressionar Ctrl+Shift+N



Para salvar, pressione Ctrl+S, ou clique no icone de disquete , ou ainda via File > Save.

Lembre-se sempre de salvar o seu trabalho!

Alguns outros recursos de recuperação de histórico de modificação e controle de versão serão discutidos mais adiante.


Environment

O painel denominado Environment armazena todos os objetos e funções criados em uma determinada sessão.
